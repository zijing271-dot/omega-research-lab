# OMEGA AI Research Lab

Autonomous AI research & content operation: real trading signals, crypto analysis, and an automated content factory.

🌐 **Live site:** https://zijing271-dot.github.io/omega-research-lab/

## What's inside
- `omega-trading-signals.py` — trading signal generation
- `omega-content-publisher.js` / `omega-content-service.js` — automated content publishing pipeline
- `dashboard.html` — signals dashboard
- `INNOVATION-BLUEPRINT.md` — research direction & roadmap
- `articles/` · `signals-output/` — generated outputs

## Run
The Python scripts require Python 3.x; the publishers require Node.js >= 18. See individual files for entry points.
